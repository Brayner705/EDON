Programacion

Aprende como crear un software utilizando un lenguaje de programacion especifico. Se cubren temas como logica de programacion, estructuras de control de flujo y algoritmos. Ademas, se proporcionan ejercicios practicos para ayudar los estudiante a poner en practica lo aprendido.

Mercadeo

Este curso de mercadeo te proporcionara las herramientas y tecnicas necesarias para crear y ejecutar estrategias de mercadeo efectivas. Aprenderas sobre investigacion de mercado, segmentacion de audiencia, planificacion de campañas, publicidad den linea, medios sociales y como medir, analizar el rendimiento de tus esfuerzos de mercadeo, este curso te ayudara a mejorar tus habilidades y aumentartus resultados.

Curador de base de datos

Aprende a organizar y limpiar tus datos para mejorar su calidad y uso en este curso practico sobre curadoria de datos.
